## Description

 A simple starter for building RESTful APIs with Nestjs, passport js, prisma, pactum and dotenv
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn db:dev:restart

## Test

```bash
# e2e tests
$ yarn db:test:restart
$ yarn test:e2e


```
